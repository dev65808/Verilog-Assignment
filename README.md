## Verilog Assignment  

### Q1   
![Question](Verilog_Q1/Question.jpg)  

- select is a 2-bit input.
- x and y are 8-bit inputs.
- z is an 8-bit output.
- Multiple conditions of select are:
    
    00: z = 0

    01: z = x + y

    10: z = x ^ y

    11: z = 0 if both x and y are odd, otherwise z = 1
